# Nova Space Telegram AI Bot

## Описание проекта

- Nova Space Telegram AI Bot - это Telegram-бот с искусственным интеллектом, созданный как интерактивное портфолио.
Бот позволяет пользователям узнать больше о создателе.
Проект объединяет обычного Telegram-бота с AI-ассистентом на базе Google Gemini, который может отвечать на вопросы и вести диалог.

## Технологии

- Python
- Aiogram 3.x
- Google Gemini API
- Telegram Bot API
- Asyncio
- Regular Expressions

## Функционал

- Команда `/start`
- Раздел "О себе"
- Мои цели
- История обучения
- Информация о менторе
- Мой прогресс
- Хобби и интересы
- Мои проекты
- GitHub
- AI-ассистент

Возможности бота:

- отвечает на вопросы пользователей
- рассказывает о портфолио
- показывает информацию о разработчике
- поддерживает свободный диалог
- использует искусственный интеллект для генерации ответов

## Запуск проекта

Клонировать репозиторий:

Перейти в папку:

cd NovaSpaceBot


Создать виртуальное окружение:

python -m venv myenv


Активировать:

Windows:

myenv\Scripts\activate


Установить зависимости:

pip install -r requirements.txt

Создать файл:

config.py

Добавить:

TOKEN = "YOUR_TELEGRAM_TOKEN"
GEMINI_KEY = "YOUR_GEMINI_API_KEY"

Запустить:

python main.py
